# holbertonschool-higher_level_programming

## Python - Import & Modules

This repository contains mandatory tasks for the Holberton School project focusing on core Python concepts: importing functions and variables from modules, creating custom modules, and handling command-line arguments.

---

### Mandatory Tasks (0 - 5)

| File Name | Description |
| :--- | :--- |
| `0-add.py` | Imports the `add` function from `add_0.py` and prints the result of 1 + 2. |
| `1-calculation.py` | Imports four basic arithmetic functions (`add`, `sub`, `mul`, `div`) from `calculator_1.py` and tests them with given variables (a=10, b=5). |
| `2-args.py` | Prints the number and list of arguments passed to the program via the command line. |
| `3-infinite_add.py` | Adds up all numerical arguments passed to the program via the command line and prints the total result. |
| `4-hidden_discovery.py` | Prints all names defined in the compiled module `hidden_4.pyc` that do not start with `__`. (This file is executed in `/tmp/`). |
| `5-variable_load.py` | Imports the variable `a` from the file `variable_load_5.py` and prints its value. |
